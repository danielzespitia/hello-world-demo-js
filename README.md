# Hello World Demo (Node.js)

This repository contains a "Hello, world!" example in Node.js.

How to run:

- With Node.js installed:

  node script.js

- With npm (uses the `start` script):

  npm install # optional if there are no dependencies
  npm start

Description:

- `script.js`: prints "Hello, world!" and some environment info.
- `package.json`: contains the `start` script.
